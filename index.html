Great Lakes Biomimicry Game

"Biomimicry is an approach to innovation that seeks sustainable solutions to human challenges by emulating nature's time-tested patterns and strategies"

I was hired by Great Lakes Biomimicry located in North East Ohio to create a polished version of the physical card game they developed. This game shows a human problem and a solution found in nature - the goal being to find how the two are connected.

While a physical card game is fun and engaging it can also be expensive for classrooms on a limited budget, the solution... use technology to create a free digital version that all classrooms can have acceess to.

National Parks Website with Weather

Site developed with HTML, CSS, JavaScript, SQL, and Spring Web MVC

Pair programming to develop a web application for the National Park Weather Service.

The program needs the ability to show a user a list of national parks, select one of those parks, and then obtain the forcast in both Fahrenheit and Celsius. The user should also be able to participate in a survey to vote on their favorite park and once submitted see the most popular parks based on those votes.
